Instructions (for the tabulator version):
1. Add/Modify lines to your liking!
2. Press COPY To TEXT AREA. This will fill the textarea below + will backup your data to a CSV file (it will be a CSV file with '|' as a delimiter* 
    2a. You can EDIT the exported CSV file (from step 2)    
    2b. Upload your EDITED csv using the upload-csv button. 
    2c. Press again the "COPY To TEXT AREA" button.
3. Press "Generate Lantern XML" button! The exported XML can be entered to Lantern.exe and will compile to ZX Spectrum,C64 etc 8-bit adventure.



* The '|' delimiter was selected for the php version of the script (not the tabulator one).This version will be available to run in your server
